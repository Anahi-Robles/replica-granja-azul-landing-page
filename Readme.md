# Granja Azul — Landing Page (réplica)

Pequeña réplica de la landing page de un restaurante, construida como proyecto estático con HTML, CSS y JavaScript.

## Descripción

Este repositorio contiene una versión local de una página de presentación para un restaurante (páginas de inicio, menú, locales, contacto, delivery, etc.). Está pensada como maqueta estática para mostrar diseño y estructura.

## Tecnologías

- HTML
- CSS
- JavaScript

## Cómo ver el proyecto

Las páginas son estáticas. Para verlas localmente basta con abrir `index.html` en un navegador. Para una experiencia de desarrollo recomendada, usa una extensión/servidor local (por ejemplo, Live Server en VS Code):

```bash
# instalar y usar Live Server (opcional)
# 1) Instala la extensión Live Server en VS Code
# 2) Abre la carpeta del proyecto y haz clic en "Go Live"
```

## Estructura principal

- `index.html` — Página principal
- `header.html`, `nav.html`, `footer.html` — Componentes fragmentados
- `Styles/` — Hojas de estilo CSS
- `img/` — Imágenes usadas en la maqueta
- `contacto.html`, `delivery.html`, `locales.html`, `nosotros.html`, `cartas.html` — Páginas secundarias
